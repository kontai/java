包與包之間的類型訪問，被訪問的包中的類必須是public的，被訪問的包中的類的方法也必須是public的
如果設置為protected,只有在被不同包繼承的子類能有訪問權限。

                  public       protected       default(默認權限）     private
    同一包中        OK              OK              OK                   OK
    同一類中        OK              OK              OK
    子類中          OK              OK
    不同包中        OK
